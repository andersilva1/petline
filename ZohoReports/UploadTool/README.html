Zoho Reports Upload Tool
Version: 10027

Zoho Reports Upload Tool is a standalone utility to upload data from your local setup behind the firewall into Zoho Reports. The tool can be used in two ways.
  1. Upload Data from CSV (Comma Separated Value) files
  2. Upload Data directly from Databases behind Firewalls
Use the tool in either way as it best suits you.
  • Method 1: If you have data (especially huge data) in CSV file format exported from your application then you could use method 1 to upload the same into Zoho Reports.
  • Method 2: If you have an application that stores data in any of the databases like MS Access, MySQL, SQL Server, Oracle or PostgreSQL running behind the firewall, you can use method 2 to upload data from such databases into Zoho Reports.
You can also setup the tool to periodically  upload (scheduling) data into Zoho Reports from your application/database.

Upload Data from CSV Files

You can bulk upload data in a CSV file into Zoho Reports using this tool. It especially useful when you need to upload more than 500,000 rows or whose file size is greater than 10 MB (this import size limit  is  applicable when you use the Zoho Reports web user interface). The tool splits the csv file into batches and uploads it to Zoho Reports, making much more resistant to intermittent Internet connectivity outages. It can be invoked in GUI as well as in Console (command line) modes.

Please find the guidelines and steps to upload data from CSV File in the below link:

http://reports.wiki.zoho.com/Upload-Tool.html

Upload Data Directly from Databases behind Firewalls

You can pull data from application databases behind firewall and upload the same into Zoho Reports. The tool directly accesses your database, queries for the required data and uploads it to Zoho Reports. Databases supported currently are MS Access, MySQL, Microsoft SQL Server, Oracle and PostgreSQL .

Please find the guidelines and steps to upload data from Databases in the below link:

http://reports.wiki.zoho.com/Upload-Tool.html
